🏠 RE-Agent — install your buttons

These buttons send listings from a page you're viewing to your local agent (which must be running: npm run dev). No extension needed.

Step 1 — show your bookmarks bar
In Chrome/Safari press ⌘ + Shift + B so the bookmarks bar is visible under the address bar.
Step 2 — drag each button up to the bookmarks bar
Grab the blue button and drop it onto your bookmarks bar. (Don't click it here — drag it up.)
📤 Send Yad2 listings    📌 Capture this post

Left button = grab every listing on a Yad2 search page. Right button = grab one selected Facebook/any post.

Step 3 — use it
Open your Yad2 search (rent, Ganei Tikva + Kiryat Ono, 4–5 rooms, up to ₪9,500) so the results are on screen, then click 📤 Send Yad2 listings in your bookmarks bar. A popup tells you how many were sent and how many triggered a WhatsApp alert.

Nothing here is automatic on its own — you click the button when you want to check. For hands-free every-5-minutes Yad2 watching, the Tampermonkey tab watcher (README → “Yad2 for free”) is the upgrade, but this button needs no install and is the fastest way to see it working.
